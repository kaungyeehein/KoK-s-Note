## Chapter 1: Get Info

```SHELL
cat hostname			# Hostname
cat etc/redhat-release	# Red hat version
cat uname				# Kernel version
cat uptime				# Uptime
cat date				# Today date
cat dmidecode | grep -i "system information" -A2	# Manufacturer & Product Name

```
